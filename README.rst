Comandos Django úteis
=====================

Instalar componentes via bower
------------------------------

.. code-block:: bash

    $ python manage.py bower_install


Gerar e comprimir arquivos css e js
-----------------------------------

.. code-block:: bash

    $ python manage.py collectstatic
